# 3. 디지털 증거

## 개요

- 본 문서는 증거의 개념, 증거능력과 증명력의 차이, 증거의 종류, 그리고 디지털 증거의 특성과 분류를 정리한 학습 문서입니다.

---

## 증거의 정의

- 사건과 관련된 사실을 입증하는 데 도움을 줄 수 있는 것
- 사건이 발생했거나 발생하지 않았음을 입증
- 범행의 의도, 알리바이 등을 입증

- 증거를 이용하여 주장하는 사실을 입증해야 함
- 반드시 **합법적인 수집 방법**이 필요함

> 증거란 법정에서 사실을 입증하기 위해 사용되는 자료이다.

---

# 증거능력과 증명력

## 증거능력

- 법정에 제출이 허용된 증거인지 여부
- 절차를 위반하면 증거능력이 부정될 수 있음

## 증명력

- 해당 증거가 사실을 얼마나 강하게 입증하는지의 정도

> 증거능력은 “제출 가능 여부”, 증명력은 “설득력의 정도”이다.

---

# 증거의 종류

## 1. 진술증거

- 사실을 목격한 사람의 진술

## 2. 물증

- 범행에 사용된 물건
- 범행 현장의 물건

## 3. 서증

- 생각이나 사실관계를 기록한 문서

---

# 디지털 증거

## 정의

- 디지털 저장 매체에 저장된 데이터 중 증거가치가 있는 데이터
- 통신망을 통해 전송되는 데이터 포함
- 아날로그 형태의 데이터도 디지털화된 경우 포함

> 디지털 증거는 전자적 형태로 존재하며 사건과 관련된 사실을 입증할 수 있는 데이터이다.

---

# 디지털 증거의 처리

- 법정에서 요구하는 절차에 따른 조사 필요
- 과학적이고 논리적인 분석 필요
- 무결성, 원본성, 관리 연속성 확보 필수

---

# 디지털 증거의 분류

## 1. 저장되는 형태에 따른 분류

### 저장매체 유형
- HDD
- SSD
- USB
- 스마트폰
- 서버 등

### 데이터 유형
- 문서
- 로그
- 이미지
- 통신기록 등

---

## 2. 생성 방식에 따른 분류

### 1) 디지털 기기에 저장된 증거 (보관 증거)

- 인간이 의도적으로 생성한 데이터
- 예:
  - 메모
  - 이메일
  - 문서 파일

### 2) 디지털 기기로 생성된 증거 (생성 증거)

- 기기 사용 과정에서 자동 생성된 데이터
- 예:
  - 로그
  - 메타데이터
  - 문서 파일 속성 정보

---

# 디지털 데이터의 저장 방식

---

## 1. 전자기적 특성

- 하드디스크 드라이브
- 플로피 디스크

### 특징

- 자화 방향을 바꿔 데이터 저장
- 강력한 자석으로 전체 데이터 삭제 가능

---

## 2. 광학적 특성

- CD
- DVD

### 특징

- 빛의 반사 유무로 데이터 저장
- 한 번 저장하면 변경이 어려움
- 데이터 보관용으로 사용
- 물리적 손상에 취약

---

## 3. 반도체 소자

### 플래시 메모리

- USB
- SD 카드
- SSD

#### 저장 방식

- 반도체 셀의 전자 존재 유무로 저장

#### 장점

- 빠른 속도
- 저전력
- 하드웨어 고장 시에도 칩 단위 데이터 추출 가능

#### 단점

- 운영체제 차원의 암호화 증가
- 암호화 우회 기술 연구 필요

---

### RAM

- 컴퓨터 주기억장치
- 휘발성 데이터 저장
- 전원 차단 시 데이터 소멸

---

# 데이터의 유형별 분류

---

## 1. 사람의 기록

- 생각과 행동을 문자, 그림으로 표현

### 디지털 변화

- 종이 → 정보 저장 매체
- 문자 → 유니코드
- 그림 → 영상 데이터

---

## 2. 문서 및 멀티미디어 데이터

- 한글, MS Office, PDF 등
- CCTV 영상
- 사진
- 동영상
- 녹음 데이터

---

## 3. 통신 데이터

- 전자메일
- 메신저 기록
- 파일 전송 기록

---

## 4. 운영체제 / 파일 시스템 데이터

- 시스템 로그
- 디렉터리 구조
- 파일 사용 흔적

→ 포렌식 아티팩트(Forensic Artifact)

---

## 5. 응용프로그램 데이터

- 웹 브라우저 기록
- 모바일 애플리케이션 데이터

---

## 6. 네트워크 로그

- 인터넷 접속 기록
- 방화벽 로그
- IDS 로그

---

## 7. 기타 기록

- 교통 기록
- 신용카드 기록
- 금융 거래 내역
- 통화 내역

---

# 보관 증거 vs 생성 증거

## 보관 증거

- 인간이 의도적으로 작성한 데이터
- 예:
  - 문서 파일 내용
  - 이메일 본문
  - 메모

## 생성 증거

- 시스템이 자동 생성한 데이터
- 예:
  - 메타데이터
  - 접근 로그
  - 파일 생성·수정 시간

> 생성 증거는 사용자가 의도하지 않았기 때문에 오히려 객관성이 높을 수 있다.

---

# 정리

- 증거는 사실을 입증하기 위한 자료이며 합법적 수집이 필수이다.
- 디지털 증거는 전자적 형태로 존재하며 다양한 저장 방식과 유형을 가진다.
